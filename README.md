# herald-webservice

小猴偷米 2018 WebService3 后端试验品，使用 Python3 + Tornado 构建；目前暂时只有框架。

## 部署

使用以下的命令来配置环境。

```bash
cd path/to/herald-webservice
pip3 install tornado
python3 app.py
```
